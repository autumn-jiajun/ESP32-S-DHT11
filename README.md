# ESP32-S-DHT11
核心功能（基于 src/main.cpp）
DHT11 温湿度采集
使用 GPIO21 连接 DHT11 传感器，每 2 秒读取一次温湿度数据。
若读取失败，网页会显示 "读取失败" 提示。
WiFi 网络功能
同时开启 AP 模式和 STA 模式：
AP 模式：创建名为 ESP32-GPIO 的热点，密码 12345678，供设备直接连接。
STA 模式：尝试连接指定路由器（默认 hpwifi，密码 hpwifi5511），连接成功后获取局域网 IP。
网页界面会显示当前 WiFi 连接状态（AP/STA 的 IP 地址）。
GPIO 控制
支持控制多个 GPIO 引脚（默认包含 2、3、4、12-19 等引脚），可通过网页按钮切换引脚高低电平（开启 / 关闭）。
网页实时显示各引脚当前状态（开启 / 关闭）。
网页交互界面
提供简洁的 HTML 网页，包含：
WiFi 状态区域（连接状态、IP 地址）。
温湿度显示区域（实时数据或错误提示）。
GPIO 控制区域（每个引脚对应 "开启"" 关闭 " 按钮）。
开发配置（platformio.ini）
开发平台：espressif32（ESP32 系列）
开发框架：arduino
依赖库：adafruit/DHT sensor library（版本 ^1.4.4，用于驱动 DHT11 传感器）
串口波特率：115200
